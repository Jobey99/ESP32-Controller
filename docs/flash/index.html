ESP32 AV Controller

Web-based AV discovery & control tool

⚡ Install Firmware

Plug in your ESP32 via USB, click the button below, and follow the prompts. No development tools needed.

  1. Connect your ESP32 to this computer via USB
  2. Click "Install" below
  3. Select the serial port when prompted
  4. Wait for the flash to complete (~30 seconds)
  5. Connect to WiFi AP "ESP32-AV-Tool" and visit 192.168.4.1
⚠️ Browser requirement: Web Serial is supported in Chrome, Edge, and Opera on desktop only. Safari and Firefox are not supported.
